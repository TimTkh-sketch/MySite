# Запуск платформы

## 1. Запустить базу данных (Prisma Dev Server)

В отдельном терминале:
```bash
cd ecom-platform
npx prisma dev
```

Оставить запущенным — он стартует локальный PostgreSQL.

## 2. Применить схему и заполнить тестовыми данными

```bash
npx prisma db push
npm run db:seed
```

Создаётся:
- Администратор: `admin@myshop.ru` / `admin123`
- Магазин "Гаджет Маркет" со slug `gadget-market`
- 5 категорий

## 3. Запустить приложение

```bash
npm run dev
```

## Открыть в браузере

- **Админка:** http://localhost:3000/admin
- **Витрина магазина:** http://localhost:3000/store/gadget-market

## Что уже работает

### Админка
- Управление магазинами (создание, редактирование, настройки)
- Управление товарами (CRUD, фото, цены, остаток)
- Управление категориями (вложенные категории)
- Управление заказами (просмотр, фильтр по статусу)
- Управление баннерами (слайдер на главной)

### Витрина
- Главная страница с баннерами, категориями, хитами продаж
- Каталог с фильтром по категориям и пагинацией
- Страница товара с галереей, описанием, похожими товарами
- Корзина с изменением количества
- Оформление заказа (имя, телефон, адрес, комментарий)

## Добавить новый магазин

1. Войти в админку → Магазины → Создать магазин
2. Добавить категории
3. Добавить товары
4. Открыть: http://localhost:3000/store/{slug}
